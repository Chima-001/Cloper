Devnest.Chima — Site Plan

WDD 131 • Chimamkpam Nnaji

SITE NAME

Devnest.Chima

"Devnest" combines "dev", short for developer, and "nest", meaning a personal home base. It represents the space where my development work lives. The .Chima extension makes it uniquely personal and identifiable as mine.

SITE PURPOSE

Devnest.Chima is a personal developer portfolio for Chimamkpam Nnaji, a software development student at BYU Pathway Worldwide and freelance web developer based in Nigeria. The site serves as a central hub where potential clients, employers, and collaborators can learn about my background, explore completed projects, assess my technical skills, and make contact.

SCENARIOS

  • What technologies and programming languages does this developer know?
  • Can I see examples of real projects with live demos?
  • How do I contact this developer for freelance work?
  • Is this developer currently available for hire?
